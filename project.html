Project 1:

Objective:

To build a responsive and informative website for a school, where users (students, parents, or staff) can get access to all academic and administrative information easily.

Overview:

The School Website was designed to be a multi-functional web application that showcases key school-related information. The platform allows users to view event details, faculty information, academic resources, and includes features like online admission forms and feedback submission.
